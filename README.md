
  # Buffet ordering app

  This is a code bundle for Buffet ordering app. The original project is available at https://www.figma.com/design/tV655RJcJETMhL39qCPc6s/Buffet-ordering-app.

  ## Running the code

  Recommended: use `pnpm` (project contains `pnpm-workspace.yaml`). If you don't have `pnpm` installed, enable Corepack (Node 16.14+/comes with Corepack) or install pnpm globally.

  Install dependencies with pnpm:

  ```bash
  corepack enable
  pnpm install
  ```

  Or with npm (if you prefer):

  ```bash
  npm install
  ```

  Start the development server:

  ```bash
  pnpm dev
  # or
  npm run dev
  ```
